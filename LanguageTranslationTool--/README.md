# Language Translation Tool

## Description
This project is a Language Translation Tool developed using Python, Streamlit, and Google Translate API. It allows users to enter text and translate it into multiple languages.

## Features
- Translate text into different languages
- Simple and user-friendly interface
- Fast and accurate translations

## Technologies Used
- Python
- Streamlit
- Google Translate API

## Installation

Install the required packages:

pip install -r requirements.txt

## Run the Application

streamlit run app.py

## Project Structure

CodeAlpha_LanguageTranslationTool

- app.py
- requirements.txt

## Output

The application translates the entered text into the selected target language and displays the translated result.
