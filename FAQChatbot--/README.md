# FAQ Chatbot

## Description
This project is a FAQ Chatbot developed using Python, Streamlit, and Scikit-learn. It answers frequently asked questions related to Artificial Intelligence and related topics.

## Features
- Interactive chatbot interface
- Answers AI-related questions
- Fast response using text similarity

## Technologies Used
- Python
- Streamlit
- Scikit-learn

## Installation

pip install -r requirements.txt

## Run the Application

streamlit run faq_bot.py
